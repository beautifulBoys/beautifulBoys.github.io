# 参数

# gutter

  • 类型：Number
  • 默认值：0
  • 描述：间距。

# align

  • 类型：String
  • 默认值："top"
  • 描述：水平分布模式，同align-items属性。

# justify

  • 类型：String
  • 默认值："center"
  • 描述：垂直分布模式，同justify-content属性。

# className
暂未实现

  • 类型：String
  • 默认值："xin-row-content"
  • 描述：该属性暂未实现。
