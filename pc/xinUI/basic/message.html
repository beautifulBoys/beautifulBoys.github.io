# Message 消息提示

# 引用

import { Message } from 'xin-ui'

# 示例

# 参数

# type

  • 类型：String
  • 默认值："info"
  • 描述：提示状态。

# message

  • 类型：String
  • 默认值：""
  • 描述：提示消息。

# closable

  • 类型：Boolean
  • 默认值：false
  • 描述：是否显示关闭按钮。

# duration

  • 类型：Number
  • 默认值：3000
  • 描述：消息自动关闭时间，单位毫秒。

# offsetTop

  • 类型：Number
  • 默认值：70
  • 描述：显示条距离上面高度。

# 事件

无事件。

# 依赖情况

无依赖。
