# 开发

# 说明

按照VuePress官网介绍，请NPM安装依赖会生成错误的依赖树，建议采用YARN安装；本文档用NPM示例启动及构建。

# 克隆仓库

git clone https://github.com/beautifulBoys/xinUI-vue.git

# 安装依赖

yarn

# 组件开发

npm run dev

# 组件构建

npm run build

# 文档站开发

npm run dev-docs

# 文档站构建

npm run build-docs
